# Memes Image Generator

Create custom memes instantly using AI! This web app generates meme images from your text prompts using Stable Diffusion. Choose from different styles like anime, photo, or watercolor.

## Features

- Text-to-image generation
- Multiple style options
- Batch image creation
- One-click downloads
- GPU acceleration support

## Quick Start

1. Install requirements: `pip install -r requirements.txt`
2. Run: `streamlit run app.py`
3. Enter prompt and generate!

Perfect for those who want to create memes.
